# Roots212 Salon AI Assistant

AI-powered Hair Salon Assistant built with Streamlit, Groq, Sentence Transformers, and FAISS using RAG.

## Installation

```bash
pip install -r requirements.txt
streamlit run root.py
```

## Environment Variables

Create a .env file locally:

```env
GROQ_API_KEY=your_groq_api_key_here
```

For Streamlit Cloud, add the key in App Settings → Secrets.
